***배열***
1. Arrays.toString(arr)
	arr을 String 처럼 출력
	[arr[0], arr[1], arr[2], ... ]

2. String.toCharArray()
	String을 char형 배열로 변환

3. String.toUpperCase()
	String을 대문자로 변환

4. Arrays.sort(arr)
	arr을 오름차순으로 정렬

5. Arrays.copyOf(원본 배열, 복사할 길이)

***String***
1. string.charAt(index)
	string의 index에 있는 문자를 출력

2. string.toCharArray()
	string을 char 타입의 배열로 변환

3. string.indexOf()
	string에서 입력한 문자의 index를 찾아서 출력
	없으면 -1

***Math***
1. Math.random()
	0 <= x < 10 사이 랜덤 생성
	0 <= x < 10 랜덤 정수 만드려면 *10

2. Math.cos()
	안의 값은 radian

3. Math.toRadians()
	deg 넣으면 radian으로 변환

2. BufferedWriter
	BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
	String s = "abcdefg";
	bw.write(s); // 버퍼에 있는 값 모두 출력
	bw.flush(); // 남아있는 데이터를 모두 출력시킴
	bw.close(); // 스트림을 닫음
